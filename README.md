#Thala 7 Sneaker Store

## Project Description
Thala 7 Sneaker Store is a front-end e-commerce website built using HTML and CSS.
It showcases a complete shopping experience including homepage, product display, shopping cart, and checkout page.

This project is designed to simulate a real-world online sneaker store with a clean and modern UI.

## Features

**Homepage**
- Responsive navigation bar with logo and menu
- Search bar, cart icon, and profile icon
- Hero section with promotional content
- Service highlights (reviews, customers, quality, rating)

**Product Display**
- Product image and detailed description
- Price and discount information
- Quantity selector
- Add to Cart button

**Gallery Section**
- Multiple product images displayed in a row
- Clean and minimal design layout

**Shopping Cart**
- Table-based cart layout
- Product name, quantity, and price
- Remove item option (X)
- Coupon code input
- Price summary (Subtotal, Taxes, Discount, Total)
- “Pay Now” button
- Payment logos

**Checkout Page**
- Contact details form (Email input)
- Multiple payment options:
- Card
- Crypto
- Bank
- Card details form:
- Card Number
- Expiry Date
- CVC
- Country & ZIP
- Order summary section
- Final payment button

##  How to run it 
1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html** or open it in any web browser.
